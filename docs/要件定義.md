# 分数マスター - 要件定義書

## 1. 概要
6歳の子供向けの分数学習ゲーム。視覚的な分数表現と段階的ヒント機能により、楽しく分数の概念を学べるWebアプリケーション。

## 2. 現在実装済み機能

### 2.1 基本ゲーム機能
- **3段階レベル**: 初級・中級・上級の難易度設定
- **固定問題数**: 各ゲーム5問で完結
- **問題プール**: 各レベル10問からランダム5問選択
- **問題タイプ**: 比較問題、等価問題、仲間はずれ問題

### 2.2 ヒント機能
- **段階的ヒント**: 数字表示 → 10秒後にヒント有効化 → 視覚表示
- **視覚的サポート**: 円形、長方形、液体の3種類の表現方法

### 2.3 フィードバック機能
- **即時フィードバック**: 正解・不正解の視覚的表示
- **間違い復習**: 不正解問題の詳細レビューと比較表示
- **視覚的比較**: 一つのグラフで2つの分数を重ねて比較表示

### 2.4 得点・記録機能
- **得点システム**: ヒント使用の有無による差別化
- **総合記録**: 正答率、総時間、問題別成績
- **成績表示**: ゲーム終了時の詳細結果

### 2.5 音声・効果音機能 ✨NEW
- **クリック音**: ボタン・選択時の効果音
- **正解音**: ピンポン音による正解フィードバック
- **不正解音**: ブザー音による不正解フィードバック
- **ヒント音**: やわらかいチャイム音
- **完了音**: ゲーム終了時のファンファーレ

### 2.6 UI/UX機能
- **レスポンシブ対応**: PC・タブレット・スマートフォン対応
- **ふりがな表示**: 全ての漢字にふりがな付き
- **アニメーション**: 軽微なトランジション効果
- **クレジット表示**: ZEAL BOOT CAMP制作者情報
